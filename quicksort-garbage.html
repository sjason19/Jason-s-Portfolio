Quicksort Garbage

I built this project out of an interest to see what some of the newest recognition software API's have to offer. I built this using:

  • Java
  • Android Studio
  • Clarifai Image Recognition API

Quicksort Garbage is an Android application where you can take a picutre of your trash and determine whether it goes into the garbage, recycling or the organics bin. This application uses the Clarifai Image Recognition API, which provides confidence recognition tags and I classify the top tags in a category as to what type of waste is behind the photo.
