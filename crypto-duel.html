Crypto Duel

I built this project because I wanted to build an exciting environment for my friends and I to compete against eachother in crypto-currency trading.

  • ReactJS
  • Firebase

Crypto Duel is a fantasy sport like crypto currency trading simulation web application that gamifies crypto currency trading. This application pulls in real crypto currency meta data to accurately simulate the crypto currency markets. Firebase is also utilized as a layer for user authentication and state management. This project is still on-going development as I am still adding additional features.
