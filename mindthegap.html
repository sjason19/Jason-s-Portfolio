Mind the Gap

I built this project as part of my final requirement for a software development course at UBC:

CPSC 210 - Software Contruction

Mind the gap is an Android application where you can view trains lines on an interactable map and the schedule for all the trains at any station. This application uses the Transport for London Unified API, which provides data for the arrival/departure information, routes and nearby stations.
